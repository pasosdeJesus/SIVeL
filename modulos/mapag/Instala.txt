Módulo Mapa con GoogleMaps por Luca Urech <lucaurech@yahoo.de>

1. Active en PHP posibilidad de abrir URLs en php.ini:

	allow_url_fopen = On

2. En el archivo de configuración de SIVeL  de su sitio (conf.php) agregue 
   modulos/mapag entre los módulos
		$modulos="modulos/mapag";
   Aumente el limite de casos por mostrar en consulta web para que puedan 
   presentarse todos los de la base.

3. Desde un navegador abra la página de actualización de la base, por
   ejemplo desde el mismo servidor con:
	https://127.0.0.1/actualiza.php  

4. Instale librerías requeridas por este módulo (que no son de dominio
   publico):
   	cd modulos/mapag/vendor
	git clone https://github.com/googlemaps/js-info-bubble.git
	git clone https://github.com/googlemaps/js-marker-clusterer.git
	
5. Puede ver el mapa desde el menú Reportes o con un URL de la forma
	https://127.0.0.1/opcion.php?num=300

Una vez en operación puede configurar algunos detalles en el archivo
conf.php de su sitio antes de la línea donde especifíco el módulo (paso 2).

$GLOBALS['mapag_fechadesde'] permite especificar una fecha inicial fija 
        (el usuario la podrá variar tras primer despliegue)

$GLOBALS['mapag_fechahasta'] permite especificar una fecha final fija
        (el usuario la podrá variar tras primer despliegue)
